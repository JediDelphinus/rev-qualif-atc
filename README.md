# REV QUALIF - Revision ATC Maroc

Application HTML statique pour la revision des qualifications ATC :

- Qualification radar
- Qualification aux procedures
- Qualification instructeur

## Publication GitHub Pages

Le fichier principal est `index.html`. GitHub Pages l'ouvrira automatiquement a la racine du site.

## Fonctionnement

- Aucun backend
- Aucune dependance externe
- Donnees integrees localement dans `index.html`
- Sauvegarde locale dans le navigateur via `localStorage`

## Avertissement pedagogique

Cet outil est un support de revision. Les reponses doivent toujours etre confirmees avec les documents officiels en vigueur : Doc 4444, Manex CCR Agadir, LOA applicables et instructions nationales applicables.

Les questions marquees "a confirmer" ou "reference manquante" ne doivent pas etre utilisees comme reference operationnelle sans verification documentaire.
